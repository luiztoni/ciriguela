# Ciriguela - backend RestAPI

 📖 Dependencias de projeto:

 * Java 17 /SDK
 * Maven 3.5+ /Build
 * Spring Boot MVC
 * Spring Data JPA
 * Spring Security
 * MariaDB JDBC Driver
 * JJWT
 * SpringDoc

## Como usar?

 1. Gerenciar configurações no arquivo resources/application.properties
 2. Iniciar a aplicação com: *mvn spring-boot:run*
 3. Acessar http://localhost:8080/swagger-ui/index.html para ver os endpoints
